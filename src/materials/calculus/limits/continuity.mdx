# Continuity of a Function

> #theorem#
> Let $f$ be a function defined on an open interval containing all values of $x$ close to $a$. Then,  
> the function $f$ is continuous at $a$ if $\lim\limits_{x\rightarrow 0}f(x)=f(a)$.

There are other specific terms to describe continuity:

- $f$ is continuous from the left at $a$ if $\lim\limits_{x\rightarrow 0^-}f(x)=f(a)$
- $f$ is continuous from the right at $a$ if $\lim\limits_{x\rightarrow 0^+}f(x)=f(a)$
- $f$ is continuous on an open interval $(a, b)$ if $f$ is continuous for every number $k$, where $a < k < b$
- $f$ is continuous on a closed interval $[a, b]$ if $f$ is continuous for every number $k$, where $a \le k \le b$

# Discontinuity of a Function

There are three types of discontinuity:

## Jump Discontinuity

![Jump Discontinuity](/materials/jump_discontinuity.png "Jump Discontinuity")

This happens when $\lim\limits_{x\rightarrow a^-} f(x) \neq \lim_{x\rightarrow a^+} f(x)$

---

## Infinite Discontinuity

![Infinite Discontinuity](/materials/infinite_discontinuity.png "Infinite Discontinuity")

This happens when the function approaches infinity or negative infinity, whether from the left side or the right side.

---

## Removable Discontinuity

![Removable Discontinuity](/materials/removable_discontinuity.png "Removable Discontinuity")

This happens when $\lim\limits_{x\rightarrow a} f(x) = f(a)$, or $f(a)$ does not exist.
